1.1.前提条件


前提条件を以下に示す

  • ナレッジベース検索ツール

    • ナレッジベース検索ツールの作業状況ビューより以下のタスクが完了していること。

      1. 目視確認項目の確認

      2. ヒアリング確認項目の確認

      3. 不明項目(ライン数が不明となっている項目)の確認

      4. SE手動算出項目(ライン数をSEが設定する項目)の確認

      Warning

      作業状況ビューにより、残項目が存在しないことを確認してください

      Figure 1. 作業状況ビュー


    • ナレッジベース検索ツールの作業状況ビューのタスクがすべて完了後、「チェック済みの移植不要項目の削除」を実行済みであること

      Figure 2. ナレッジベース検索ツールの「チェック済みの移植不要項目の削除」ボタンについて


    • 「CSV保存ボタン」を実行し、ファイル出力すること.本例ではknowhow_result.csvとして出力する

      Figure 3. ナレッジベース検索ツールの「CSV保存」ボタンについて


  • 依存性検索ツール

    Figure 4. 依存性検索検索結果ビューより「CSV出力」を実施していること。本例ではdepends_result.csvとして出力する




1.2.レポート出力方法


  1. ナレッジベース検索ツールまたは、依存性検索ツールでCSV出力した２つのファイルをEclipseのプロジェクト内にコピーしてださい

    Important

    ナレッジベース検索ツールの結果をレポートする場合は、eclipse/plugins/tubame.portability-*/resources/xml/checkListInformation_ja.xmlも必要となるため、CSVファイルに加え、checkListInformation_ja.xmlファイルもEclipseのプロジェクト内の同じ場所にコピーしてださい

    Important

    汎用検索検索ツールの結果をレポートする場合は、前提条件にあるように、かならずCSV出力したものをEclipseのプロジェクト内にコピーしてださい。

    Note

    ここではナレッジベース検索ツールで出力したファイルをknowhow_result.csv、依存性検索ツールで出力したファイルをdepends_result.csvとして保存する

    保存先はプロジェクト内であれば、任意ですが、この例ではreport_inputディレクトリにCSVファイルを保存する

  2. CSVファイルの拡張子を変更する

    Note

    ナレッジベース検索ツールで出力したCSVファイルの拡張子をjbmに変更する( knowhow_result.csv -> knowhow_result.jbm )

    依存性検索ツールで出力したCSVファイルの拡張子をgjbmに変更する( depends_result.csv -> depends_result.gjbm )

  3. TubameReportGenerator.xmlを取得する

    Note

    TubameReportGenerator.xmlについては以下より取得可能です

    https://github.com/TUBAME/migration-knowledge/releases/download/report-generator/tubame-report-knowledge.zip

  4. ナレッジXMLにTubameReportGenerator.xmlを指定して、ナレッジベース検索を実行する。本例の場合は、report_inputディレクトリを右クリックして、ナレッジベース検索を実行する

    Figure 1. ナレッジベース検索メニュー


    Warning

    以下の点に注意してください

    • 検索対象に必ずナレッジベース検索ツールまたは依存性検索ツールで出力したCSVファイル(本例ではknowhow_result.jbm及びdepends_result.gjbm)が格納されていることを確認してください

    • ナレッジベース検索ツールの結果をレポートする場合は、eclipse/plugins/tubame.portability-*/resources/xml/checkListInformation_ja.xmlも必要となるため、CSVファイルに加え、checkListInformation_ja.xmlファイルもknowhow_result.jbmと同じ場所に配置されていることを確認してください

    • ナレッジXMLはTubameReportGenerator.xmlを指定してください

    • 検索対象は、レポート対象のjbmやgjbmファイルが存在するディレクトリを指定してください。本例の場合は、report_inputディレクトリを右クリックして、ナレッジベース検索を実行する

    • 結果出力先ファイルは上書きされないように出力先のファイル名はreport_search_result.jbm等に変更するようにしてください。

  5. 出力されたレポートの確認

    Note

    レポートはデフォルトではeclipse\plugins\tubame.knowhow-*/report配下に出力されます。 出力先を変更したい場合は、TubameReportGenerator.xmlをナレッジ管理ツールで開き、チェックアイテム「search-csv」の検索キーワード1に出力先ディレクトリをフルパスで指定してください

    Figure 2. TUBAMEレポートイメージ


    Important

    レポート出力後に再度jbmファイルエディタで参照、編集する場合は、プロジェクト内にコピーしたファイルのcheckListInformation_ja.xmlをeclipse/plugins/tubame.portability-*/resources/xml/配下にコピーする必要があります
