2.動作環境


2.1.ソフトウェア要件


TUBAMEマイグレーションツールは、Eclipse Pluginとして動作します。以下のソフトウェア環境が必要となります。

Table 1. ソフトウェア要件

項目要件
OSWindows, MacOS
JavaOracle JRE1.6以上
EclipseEclipse3系、または、Eclipse4系




3.インストール


3.1.自動インストール


自動インストールは、TUBAME（Eclipse Plugin)のインターネットからのダウンロード、および、Eclipseへのインストールを自動で実行する方法です。 ソフトウェア要件を満たしていることを確認し、以下の手順を実施します。

  1. Eclipse起動

  2. Eclipseのメニューより Help → Install New Software... を選択し、Install画面を起動します。起動後、Addボタンをクリックします。

    Figure 1. Install画面


  3. AddReposiry画面にて以下の項目を入力し、OKボタンをクリックします。

    Table 1. AddReposiry画面入力項目

    NameLocation
    TUBAMEhttp://dl.bintray.com/tubame-developers/tubame-release/

    Figure 2. AddReposiry画面


  4. インストール画面に表示される「TUBAME」チェックボックスをチェックし、「Next」ボタンをクリックします。

    Figure 3. 


  5. インストール詳細画面が表示されるので、「Next」ボタンをクリックします。

    Figure 4. 


  6. ライセンスレビュー画面が表示されるので、「I accept ...」をチェックし、「Finish」ボタンをクリックします。

    Figure 5. 


    Note

    インストール完了後にEclipseを再起動します。再起動後に以下を実施しTUBAME Eclipse Pluginがインストールされていることを確認してください。

    Eclipseのメニューより Help → Install Details を選択し、Install Details画面にて、Pluginsタブを選択し、「tubame」を入力して、3つのpluginが表示されることを確認します。

    Figure 6. Install Details画面




3.2.手動インストール


手動インストールは、TUBAME(Eclipse Plugin)のリリースページからのダウンロード、および、Eclipseへのインストールを手動で実施する方法です。 ソフトウェア要件を満たしていることを確認し、以下の手順を実施します。

  1. TUBAMEダウンロード

    https://github.com/TUBAME/migration-tool/releases/download/v1.1.0/tubame_110.zip

    Note

    上記URLは2016/02/18時点の最新版である、TUBAME1.1.0のダウンロード例です。最新版のダウンロードについては、以下より最新版を確認して、インストールしてください。

    • https://github.com/TUBAME/migration-tool/releases/latest

  2. TUBAME Eclipse Pluginのインストール

    ダウンロードファイルを展開するとplugins フォルダ配下に3つフォルダが存在しています。

    • tubame.knowhow_*

    • tubame.portability_*

    • tubame.wsearch_*

    この3つのフォルダをコピーして、Eclipseをインストールしたディレクトリ配下にあるpluginsディレクトリ配下に配置します。

    Figure 1. Eclipseのpluginsディレクトリ


インストールは上記で完了です。

Note

インストール完了後は、Eclipseを起動します。起動後は以下を実施し、TUBAME Eclipse Pluginがインストールされていることを確認してください。

Eclipseのメニューより Help → Install Details を選択し、Install Details画面にて、Pluginsタブを選択し、「tubame」を入力して、3つのpluginが表示されることを確認する
