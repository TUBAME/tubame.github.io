What is TUBAME?
TUBAME is a tool that allows the calculation of the correction effort and extraction of corrections necessary when migrating between JBoss or JBoss from a commercial AP server.
It contains a multiple sub-tools, you can use it as an Eclipse plug-in.
Please utilize by all means because it is published in TUBAME community as OSS product.
